# VariablesYOperadores

¿Para qué se utilizan los operadores lógicos?

Los operadores lógicos permiten combinar condiciones en una expresión booleana. Se utilizan para controlar el flujo del programa mediante estructuras condicionales como if-else, permitiendo evaluar múltiples condiciones al mismo tiempo.

¿Por qué es importante declarar correctamente el tipo de dato?

Es importante porque el tipo de dato determina qué tipo de información puede almacenar una variable y cómo se comportará en operaciones. También influye en el uso de memoria y en la precisión de los resultados, especialmente en operaciones matemáticas.
